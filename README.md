##  Getting Started

### Prerequisites

Make sure you have installed:

* Node.js
* npm or yarn

### Installation

```bash
git clone https://github.com/riyakarnan2008-hue/purple.git

cd purple

npm install
```

### Run the Project

```bash
npm start
```

or

```bash
npm run dev
```
